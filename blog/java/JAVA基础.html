# Java基础

# Java版本

  • Java SE：Java标准版，定位在PC使用
  • Java EE：Java企业版，定位在服务器端使用
  • Java Me：Java微型版，定位消费型电子产品

# 特性

  • 跨平台、可移植性

    Java的核心优势，注重跨平台特性和可移植性。

  • 安全性

  • 面向对象

    完全的面向对象语言。

  • 简单性

    Java是C++的简化版，有人称作"C++-"，指的就是将C++的一些特性去掉，比如头文件、指针运算、结构等等。

  • 高性能

    Java在发展过程中通过虚拟机提高了几十倍的效率，比如通过JIT即时编译技术提高运行效率。

  • 分布式

  • 多线程

    对线程可以带来更好的交互响应和实时行为，Java多线程的简单性是Java成为主流服务器端开发语言的主要原因之一。

  • 健壮性

# JDK、JRE、JVM

# JVM

编译器将Java语言编译成字节码，之后字节码被送到JVM执行，JVM跟操作系统挂钩，将命令给操作系统执行。

javac Hello.java //该命令编译Java语言，会在目录下生成Hello.class文件

之后在虚拟机中运行代码

java Hello  //在虚拟机中执行字节码

# JRE

JRE包含以下内容：

  • JVM
  • Java本身的一些库函数
  • 运行Java应用程序必须的相关文件

# JDK

JDK是以下内容的合集：

  • JRE
  • Java编译器
  • Java调试器
