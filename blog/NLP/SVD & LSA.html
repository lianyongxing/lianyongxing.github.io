# SVD & LSA

# 方阵的分解

Ax=λxAx = \lambda x Ax=λx

矩阵AAA是nnn维方阵，xxx是nnn维向量，λ\lambdaλ是矩阵AAA的一个特征值，xxx是矩阵AAA的特征值λ\lambdaλ对应的特征向量。计算出特征值和特征向量后，就可以将矩阵AAA进行特征分解，计算所有特征值和对应的特征向量，可以表示为

A=WΣW−1A=W\Sigma W^{-1} A=WΣW​−1​​

其中WWW是这nnn个特征向量所张成的nnn维方阵，而Σ\SigmaΣ是这n个特征值为主对角线的nnn维方阵。

但是，按此方法进行分解，要求AAA必须是方阵，如果不是方阵，则通过SVD进行分解。

# SVD分解

SVD也是矩阵进行分解，但是和特征分解不同，SVD不要求矩阵为方阵，假设矩阵为m∗nm*nm∗n的矩阵，则可以定义SVD分解为

A=UΣVTA=U\Sigma V^{T} A=UΣV​T​​

其中UUU是m∗mm*mm∗m的矩阵，Σ\SigmaΣ是m∗nm*nm∗n的矩阵，主对角线上是奇异值，VVV是n∗nn*nn∗n的矩阵，且UTU=IU^{T}U=IU​T​​U=I，VTV=IV^{T}V=IV​T​​V=I。

将VTV^{T}V​T​​和VVV相乘，即可得到一个方阵，因此对其进行特征分解，如下：

(ATA)ui=λiui(A^{T}A)u_{i} = \lambda_{i}u_{i} (A​T​​A)u​i​​=λ​i​​u​i​​

此时可以得到对应的nnn个特征值和特征向量，将其张成n∗nn*nn∗n的矩阵vvv，则是上面的VVV矩阵，同理VVTVV^{T}VV​T​​可以通过特征分解计算UUU矩阵。

# 主题模型LSA的例子

目标：有n句话，来从中提取存在可能哪些主题。

每句话用TFIDFVector或者CountVector进行表示，构造稀疏矩阵。比如存在200句话，BOW中存在1024个单词，则可以用矩阵(200, 1024)进行表示，之后对其进行SVD分解，得到U$矩阵(200, 200)表示200个样本，对应了200个topic， Σ\SigmaΣ矩阵(200, )表示200个表征topic分数的奇异值和VVV矩阵(200, 1024)表示200个topic对应1024个vocab，这个VVV是经过转置后的。

此时可以取前nnn个奇异值来表征topic的信息，也可以用VVV矩阵的topic所对应的vocab来表征每个topic的关键词。
